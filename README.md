# POLR Holographic Viewer

Holographic Construction Detail Viewer with Semantic Compression

## Overview

Transform construction shop drawings into interactive 3D holograms using semantic compression technology.

**Compression Ratio:** ~5,000:1
- Traditional: 50 MB volumetric data
- Semantic: 500 bytes parameters â†’ Full 3D reconstruction

## Features

- **45 Construction Details** - Air Barrier, Waterproofing, Roofing
- **Holographic Effects** - Glow, scan lines, flicker
- **WebXR Support** - AR/VR for Quest, Vision Pro, HoloLens
- **Looking Glass** - True holographic display support
- **GCP Products** - BITUTHENE, PERM-A-BARRIER callouts

## Quick Start

```bash
cd polr-holographic-viewer
npm install
npm run dev
```

Open http://localhost:3000
